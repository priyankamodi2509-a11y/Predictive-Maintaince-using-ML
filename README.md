# Predictive Maintenance App 🏭

A simple Machine Learning web app built using Streamlit that predicts machine failure using sensor data.

---

## 📌 Problem Statement
To predict machine failure so that maintenance can be done before breakdown.

---

## 🧠 Model Used
- Random Forest Classifier

---

## ⚙️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

---

## 📁 Files
- app.py → Streamlit app
- rf.pkl → trained model
- requirements.txt → dependencies

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
